# Collapsible Work & Projects Section

## Цель

Сделать секцию «Work & Projects» компактнее за счёт двухуровневого сворачивания: блоки компаний и проекты внутри них.

## Требования

### Поведение

1. **Двухуровневое сворачивание:**
   - **Уровень 1 — блоки компаний** (Nadeks, Mahuru, Freelance, Pet Projects): кликабельный заголовок с chevron-стрелкой (▶ свёрнуто / ▼ развёрнуто)
   - **Уровень 2 — проекты внутри блока**: каждая карточка проекта сворачивается до однострочного заголовка с chevron-стрелкой

2. **Начальное состояние при загрузке:** всё свёрнуто. Видны только заголовки блоков компаний.

3. **Множественное раскрытие:** несколько блоков и проектов могут быть открыты одновременно (не аккордеон).

4. **Состояние не персистится** — при перезагрузке страницы всё снова свёрнуто.

### Визуальный стиль

Дизайн должен максимально сохранить текущий стиль карточек проектов. Изменения минимальны:

1. **Заголовок блока компании (свёрнутый):**
   - Chevron-стрелка ▶ зелёного цвета (#00d084) справа
   - Существующая вертикальная зелёная полоса и название компании
   - Роль, тип работы и период — как сейчас
   - Справа: счётчик проектов (например, «4 projects» / «4 проекта» через i18n), мелким серым текстом

2. **Заголовок блока компании (развёрнутый):**
   - Chevron-стрелка ▼ зелёного цвета
   - Всё остальное идентично свёрнутому состоянию

3. **Проект (свёрнутый):**
   - Chevron-стрелка ▶ зелёного цвета
   - Цветная точка статуса (жёлтый/зелёный/красный/серый — как сейчас)
   - Название проекта
   - Период справа (если есть)

4. **Проект (развёрнутый):**
   - Chevron-стрелка ▼ зелёного цвета
   - Полное содержимое текущей карточки проекта: описание (intro + bullets), стек технологий, все ссылки и кнопки
   - Визуально идентично тому, как карточка выглядит сейчас

5. **Блок Pet Projects:** заголовок без компании/роли (как сейчас), только «Pet Projects» + счётчик

### Анимации

- Раскрытие/сворачивание: Framer Motion `AnimatePresence` + `motion.div` с `animate={{ height: "auto", opacity: 1 }}` / `exit={{ height: 0, opacity: 0 }}`
- Chevron-стрелка: плавный поворот на 90° через `rotate`
- Существующие анимации `whileInView` для появления блоков при скролле — сохранить
- Hover-эффект карточек (`y: -4`) — применять только к свёрнутым заголовкам проектов

### i18n

Новые ключи в `lib/i18n.ts`:

| Ключ | EN | RU |
|------|----|----|
| `work_projects_count` | `{count} projects` | `{count} проектов` / склонение |
| `work_project_count_one` | `1 project` | `1 проект` |
| `work_project_count_few` | `{count} projects` | `{count} проекта` |

Примечание: для русского языка нужно склонение (1 проект / 2 проекта / 5 проектов).

### Взаимодействие

- Клик по заголовку блока → toggle раскрытия блока
- Клик по свёрнутому проекту → toggle раскрытия проекта
- Клик по ссылкам внутри раскрытого проекта (GitHub, RuStore, Play, скриншоты) → текущее поведение (открытие ссылки / модалка скриншотов / toast)
- `cursor: pointer` на всех кликабельных заголовках

- Модальное окно скриншотов — без изменений
- Цветовая кодировка статусных точек — без изменений
- Стек технологий, описания, ссылки — визуально идентичны текущим
- Навигация в Navbar — без изменений
