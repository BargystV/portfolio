# Явные кнопки действий в карточках проектов

## Проблема

Названия проектов кликабельны, но это неочевидно для пользователя. Нет визуального индикатора, что по проекту можно перейти на внешний ресурс или открыть галерею скриншотов.

## Решение

Заменить цветные точки статуса на кликабельные боковые полосы в заголовке карточки проекта. Добавить текстовые кнопки в раскрытую карточку.

## Типы действий

Каждый проект имеет ровно одно действие. Приоритет определения:

1. **Скриншоты** (`screenshots?.length`) → иконка 🖼, жёлтый цвет (`#eab308`), открывает `ScreenshotModal`
2. **Внешняя ссылка** (`githubUrl || rustoreUrl || googlePlayUrl` строка) → иконка ↗, зелёный цвет (`#00d084`), `window.open`
3. **Приватный** (`isPrivate`, без скриншотов) → иконка 🔒, красный цвет (`#ef4444`), не кликабельный
4. **Нет действия** → иконка —, серый цвет (`white/20`), не кликабельный

### Маппинг текста кнопки в раскрытой карточке

| Действие | Текст кнопки (en) | Текст кнопки (ru) |
|---|---|---|
| GitHub | `GitHub ↗` | `GitHub ↗` |
| RuStore | `RuStore ↗` | `RuStore ↗` |
| Google Play (строка) | `Google Play ↗` | `Google Play ↗` |
| Скриншоты | `Screenshots 🖼 (N)` | `Скриншоты 🖼 (N)` |
| Приватный | — нет кнопки | — нет кнопки |
| Нет действия | — нет кнопки | — нет кнопки |

## Дизайн свёрнутой карточки

```
┌──────────────────────────────────────────────────┐
│ ↗ │  Portfolio          Март 2026 – н.в.    ▶  │
│(40px)│                                           │
└──────────────────────────────────────────────────┘
```

- Боковая полоса: ширина 40px, на всю высоту строки
- Фон полосы: `rgba(цвет, 0.08)`, разделитель справа: `1px solid rgba(цвет, 0.2)`
- Иконка по центру полосы
- При hover полосы: фон ярче (`rgba(цвет, 0.15)`)
- `overflow: hidden` + `border-radius: 12px` на контейнере
- Полоса — отдельный элемент `<a>` или `<span>`, клик по ней НЕ тригерит toggle карточки (`e.stopPropagation()`)

## Дизайн раскрытой карточки

- Боковая полоса остаётся только в заголовочной строке
- Тело карточки (`border-top` разделитель) — без полосы, как сейчас
- Под стеком технологий — текстовая кнопка:
  - `border-radius: 8px`, `border: 1px solid rgba(цвет, 0.3)`, `background: rgba(цвет, 0.05)`
  - Цвет текста = цвет действия
  - `font-family: monospace`, `font-size: 12px`, `padding: 7px 16px`
